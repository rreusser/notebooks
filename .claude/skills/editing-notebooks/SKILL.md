# Overview

This guide provides comprehensive guidance for developing Observable notebooks using Notebook Kit (Observable 2.0), including debugging, reactive programming patterns, and best practices learned from production notebooks.

For authoring narrative sections and working with equations, see ./LANGUAGE.md for language and usage guidance. For Notebook Kit data model fundamentals, see ./NOTEBOOKS.md. For runtime notebook debugging instructions, see ./DEBUGGING.md.
